A comprehensive package designed to enhance developer productivity and streamline common Laravel development tasks. Provides utilities for command execution, environment management, and development workflows.

#### Features

- 🖥️ **Artisan Command Execution** - Execute artisan commands from anywhere
- 🌍 **Environment Tools** - Manage application environment settings
- 🛠️ **Development Utilities** - Useful development helpers & shortcuts
- 🔐 **Secure Configuration** - Encrypted configuration management
- 📊 **Debug Utilities** - Enhanced debugging capabilities
- 🔍 **Code Analysis** - Static code analysis tools
- 📝 **Log Viewer** - Integrated log viewing & filtering
- ⚙️ **Configuration Sync** - Synchronize configurations across environments
- 🚀 **Performance Monitoring** - Monitor application performance
- 📋 **Database Utilities** - Database management helpers

#### Installation

```bash
composer require adel-mahmoud/laravel-devtools
```

#### Tech Stack
- Laravel 10+
- PHP 8.1+
- Custom Artisan Commands
- Eloquent ORM

#### Usage Examples

```php
// Easy command execution
DevTools::executeCommand('migrate');

// Environment management
DevTools::setEnvironmentValue('APP_DEBUG', true);

// Secure configuration
DevTools::secureConfig('api_key', $value);

// Log viewing
DevTools::viewLogs('error');

// Performance monitoring
DevTools::monitorPerformance();
```
